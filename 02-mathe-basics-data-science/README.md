# Mathe-Basics für Data Science (Thomas Nield)
**Vorbereitungsmodul für den Master: Mathematik für Nachhaltigkeit, Wirtschaft und Data Science**

Dieses Repository dokumentiert die mathematischen Kernkonzepte, die für die ökonomische Modellierung und Datenanalyse im Masterstudium unerlässlich sind.

## 📚 Kapitelübersicht & Lernziele

| Kapitel | Fokus | Relevanz für den Master |
| :--- | :--- | :--- |
| **01. Analysis** | Funktionen, Ableitungen, Optimierung. | Grundlage für ökonomische Grenzwertanalysen. |
| **02. Wahrscheinlichkeit** | Bayes-Theorem, Binomialverteilung. | Risikoanalyse in Nachhaltigkeitsprojekten. |
| **03. Statistik** | Hypothesentests, Signifikanz. | Validierung von Wirtschaftsmodellen. |
| **04. Lineare Algebra** | Matrizen, Vektoren, Eigenwerte. | Strukturierung großer Datensätze (Input-Output). |
| **05. Lineare Regression** | Korrelation, KQ-Schätzung. | Vorhersage von Wirtschaftstrends. |
| **06. Klassifikation** | Logistische Regression, Odds. | Entscheidungstheorie und Marktsegmentierung. |
| **07. Neuronale Netze** | Backpropagation, Aktivierung. | Komplexe Mustererkennung in Umweltdaten. |
| **08. Ausblick** | Portfolio & Karriere. | Strategische Ausrichtung auf den Arbeitsmarkt. |

---
**Methodik:** Jedes Kapitel enthält ein Jupyter Notebook mit theoretischer Herleitung (LaTeX) und Python-Implementierung (NumPy/SciPy).
