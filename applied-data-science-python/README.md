# Applied Data Science with Python Specialization (University of Michigan)

This directory contains a systematic progression through the five-course specialization, documenting the transition from basic data manipulation to advanced social network analysis. 

## 📊 Learning Roadmap

### [01. Pandas Foundations](./01-pandas-foundations)
* **Focus:** Vectorized data manipulation and cleaning.
* **Key Skills:** DataFrames, Series, Boolean masking, and merging disparate datasets.
* **Goal:** Transforming raw, messy data into analysis-ready structures.

### [02. Visualization](./02-visualization)
* **Focus:** Information design and statistical plotting.
* **Key Skills:** `Matplotlib` subplots, `Seaborn` statistical overlays, and "Data-to-Ink" ratio optimization.
* **Goal:** Communicating evidence-based insights through high-density charts.

### [03. Machine Learning](./03-machine-learning)
* **Focus:** Predictive modeling and model evaluation.
* **Key Skills:** Scikit-learn pipelines, cross-validation, and over/under-fitting analysis.
* **Goal:** Building robust models to predict outcomes on unseen data.

### [04. Text Mining](./04-text-mining)
* **Focus:** Natural Language Processing (NLP).
* **Key Skills:** Regular Expressions (Regex), NLTK, and sentiment analysis.
* **Goal:** Extracting structured features from unstructured text data.

### [05. Social Network Analysis](./05-social-network-analysis)
* **Focus:** Graph theory and connectivity.
* **Key Skills:** `NetworkX` implementation, centrality measures, and link prediction.
* **Goal:** Analyzing the influence and structure of networked relationships.

---

## 🛠️ Technical Stack
* **Language:** Python 3.x
* **Primary Libraries:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `networkx`, `nltk`
* **Environment:** VS Code / Jupyter Notebooks
